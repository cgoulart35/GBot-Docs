### /patreon
>#### Description
>    * `In the GBot Patreon server, specify a server's ID to enable GBot functionality in that server. (patrons only)`
>
>#### Syntax
>    * `.[patreon|pt] <server_id>`
>
>#### Example
>
>    ```
>    .pt 012345678910111213
>    ```
