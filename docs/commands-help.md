### /help
>#### Description
>    * `Type .help command for more info on a command. You can also type .help category for more info on a category.`
>
>#### Syntax
>    * `.help [command]`
>
>#### Example
>
>    ```
>    .help
>    .help GTrade
>    .help cr
>    ```
